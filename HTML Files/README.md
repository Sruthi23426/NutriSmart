# 🥗 NutriSmart

NutriSmart is a full-stack web application that provides personalized nutrition and health recommendations based on a user's profile, lifestyle, and medical conditions. It helps users track their health, calculate BMI, and receive customized diet suggestions.

---

## ✨ Features

- User Registration & Login
- BMI Calculation
- Personalized Health Profile
- Disease-Specific Diet Recommendations
- Weekly Progress Tracking
- Interactive Dashboard
- Responsive User Interface

---

## 🛠️ Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
- Mongoose
